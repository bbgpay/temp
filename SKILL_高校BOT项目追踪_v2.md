# 高校BOT项目追踪技能说明

## 📋 技能概述
自动追踪全国高校校园生活服务设施BOT运营项目（直饮水、淋浴、洗衣机、吹风机），通过网络搜索获取最新招标/中标信息，自动去重后生成两个Excel文件并推送到GitHub仓库，同时发送邮件通知。

## 🎯 触发关键词
当你说以下任意内容时，我会自动执行全套流程：
- **"更新高校BOT项目表"**
- **"搜索新的高校服务项目"**
- **"高校生活服务设施运营"**
- **"BOT项目追踪"**
- **"高校直饮水/淋浴/洗衣机/吹风机项目"**

## 🔄 执行流程

### 第一步：读取已有记录
- 从 `data/campus_service_existing.json` 读取已收录的 (学校, 设施类型, 运营公司) 组合
- 用于后续去重比对

### 第二步：网络搜索新项目
使用 `web_search` 搜索：
1. "2025 2026 高校 直饮水 淋浴 BOT 中标 运营"
2. "2025 2026 高校 洗衣机 吹风机 BOT 中标结果"
3. "高校 校园生活服务 BOT 招标公告"

### 第三步：数据提取与去重
**提取字段：**
- 学校名称、设施类型、配置情况、消费场景
- 设备厂家、支付方式、运营模式、运营公司
- 项目到期时间、服务年限、联系人/电话、备注

**去重逻辑：**
- 使用 (学校名称, 设施类型, 运营公司) 三元组比对
- 只保留真正的新记录

### 第四步：生成双表
使用 openpyxl 生成两个Excel文件：

**文件1：`高校校园生活服务设施运营总表.xlsx`**
- 包含 ALL 记录（原有 + 新增）
- 序号自动重新编排
- 格式：蓝色表头、交替行颜色、自动列宽、冻结首行

**文件2：`高校生活服务新发现项目表.xlsx`**
- 仅包含本次新增记录
- 格式同上

### 第五步：更新去重数据
更新 `data/campus_service_existing.json`：
```json
{
  "last_update": "YYYY-MM-DD",
  "source": "公开招标/中标公告",
  "records": [
    {"school": "xxx", "type": "直饮水", "company": "xxx"}
  ]
}
```

### 第六步：推送到GitHub
```bash
cd /tmp/temp_repo

git commit -m "更新高校BOT项目表 - 新增X条记录，总计Y条"
```

## ⚠️ 注意事项

1. **去重逻辑**：使用 (学校名称, 设施类型, 运营公司) 三元组去重
2. **服务期筛选**：只收录"在服务期间"或"已中标待运营"的项目，已到期项目不收录
3. **数据来源**：高校官网、公共资源交易平台、采招网等公开信息
4. **GitHub路径**：仓库 `bbgpay/temp`，使用已有的 TOKEN 进行 push
5. **邮件配置**：使用已配置的163邮箱 `nerovatech@163.com` 发送
6. **定期执行**：可通过说"帮我设置每周一早上9点自动更新高校BOT项目表"来设置定时任务（定时任务会自动执行完整流程，包括发送邮件）
